This folder is for your project images.
Please upload your images here with the following naming convention:

project-<project_number>-<image_number>.<extension>

Examples:
- project-1-1.jpg (Main image for Watermelon Shop)
- project-1-2.jpg (Second image for Watermelon Shop)
- project-2-1.jpg (Main image for Watermelon Fest)
- ... and so on.

The website currently supports:
- Project 1: up to 3 images
- Project 2: up to 6 images
- Project 3: up to 7 images
- Project 4: up to 9 images

Note: Demo images from Picsum will be shown if the files are not found.
Kích thước ảnh sẽ tự động được xử lý để phù hợp với khung hình.
